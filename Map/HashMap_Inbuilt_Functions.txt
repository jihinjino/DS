
Java HashMap Inbuilt Functions
==============================

1. put(K key, V value)
   - Adds a key-value pair or updates the value if key already exists.

2. get(Object key)
   - Returns the value associated with the key, or null if not present.

3. containsKey(Object key)
   - Returns true if the map contains the specified key.

4. containsValue(Object value)
   - Returns true if the map maps one or more keys to the specified value.

5. remove(Object key)
   - Removes the key and its corresponding value from the map.

6. size()
   - Returns the number of key-value mappings in the map.

7. isEmpty()
   - Returns true if the map contains no key-value mappings.

8. clear()
   - Removes all of the mappings from the map.

9. keySet()
   - Returns a Set view of the keys contained in this map.

10. values()
    - Returns a Collection view of the values contained in this map.

11. entrySet()
    - Returns a Set view of the mappings contained in this map.

12. replace(K key, V value)
    - Replaces the entry for the specified key only if it is currently mapped to some value.

13. replace(K key, V oldValue, V newValue)
    - Replaces the entry for the specified key only if currently mapped to the specified value.

14. getOrDefault(Object key, V defaultValue)
    - Returns the value to which the specified key is mapped, or defaultValue if the map contains no mapping for the key.

15. putIfAbsent(K key, V value)
    - If the specified key is not already associated with a value (or is mapped to null), associates it with the given value.
